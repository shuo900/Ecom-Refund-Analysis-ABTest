# 🛒 电商大促退款归因诊断与 A/B Test 增长分析

## 📌 项目初衷
在年货节/情人节等大促期间，针对退款率异常波动，本项目通过整合销售数据与 5000+ 条客服文本，实现了从“异常诊断”到“策略验证”的全链路分析。

## 🛠️ 技术实现路径
1. **数据清洗 (SQL)**: 多表左连接关联订单与退款明细，计算各维度退款率。
2. **文本归因 (Python)**: 
   - 运用正则匹配提取价格敏感特征。
   - 计算 **Relative Risk (RR)**，发现 3 件组退款风险显著高于 1-2 件组。
3. **策略验证 (A/B Test)**:
   - **Power Analysis**: 预估最小样本量以保证统计功效。
   - **Z-test**: 验证转化率提升 **19.9%** ($p = 0.0033$)，具有统计显著性。
